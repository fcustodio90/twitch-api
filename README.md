# twitch-api
A fake server running a Restful API

For documentation purposes read https://github.com/typicode/json-server

## Installation

0. Clone the repo
1. Run npm install
2. Run npm start

## RESTFul convention

### List all the records

GET /streams

### Get one particular record

GET /streams/:id

### Create a Record

POST /streams

### Update a record

PUT /streams/:id

### Delete a record

DELETE /streams/:id
